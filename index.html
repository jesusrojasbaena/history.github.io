Línea temporal

De Git a GitHub

Para entender GitHub hay que empezar antes: con el problema de coordinar miles de cambios en proyectos enormes sin convertir el desarrollo en una excursión al caos.

Antes

El problema: colaborar era difícil

Los equipos necesitaban controlar versiones, evitar pérdidas de código y coordinar aportaciones de muchos desarrolladores.

2005

Linus Torvalds crea Git

Git nace para gestionar el desarrollo del kernel de Linux: rápido, distribuido y preparado para muchos colaboradores.

2008

Nace GitHub

GitHub aparece como una plataforma web para alojar repositorios Git y hacer más fácil colaborar, revisar código y compartir proyectos.

2010s

GitHub se convierte en estándar

Los repositorios, issues, pull requests y perfiles convierten GitHub en un punto central para desarrolladores y empresas.

2018

Microsoft compra GitHub

Microsoft adquiere GitHub por 7.500 millones de dólares y lo mantiene como plataforma clave para el desarrollo abierto y privado.

Hoy

GitHub es un ecosistema

Repositorios, GitHub Pages, GitHub Actions, Copilot, seguridad, automatización y colaboración en una misma plataforma.

Concepto clave

Git no es GitHub

Git es la tecnología. GitHub es la plataforma que la popularizó y la convirtió en algo más visual, social y colaborativo.

⚙️

Git

Es un sistema de control de versiones creado en 2005 por Linus Torvalds.

  • Funciona en local.
  • Guarda versiones mediante commits.
  • Permite trabajar de forma distribuida.
  • No depende de GitHub para existir.
☁️

GitHub

Es una plataforma web fundada en 2008 para alojar repositorios Git y facilitar la colaboración.

  • Guarda repositorios en la nube.
  • Permite colaborar con issues y pull requests.
  • Ofrece perfiles, comunidad y automatización.
  • Puede publicar webs con GitHub Pages.
Nombre

Qué significan Git y Hub

El nombre GitHub une dos ideas: la herramienta Git y el concepto de hub como centro de conexión. Por una vez, un nombre tecnológico tiene sentido. Casi sospechoso.

🧩

Git

Git no es un acrónimo oficial. Linus Torvalds eligió el nombre con su típico tono irónico. Técnicamente, se convirtió en el nombre del sistema de control de versiones.

🌐

Hub

Hub significa centro, núcleo o punto de conexión. GitHub puede entenderse como el centro donde se reúnen proyectos Git.

Fundación

Quién creó GitHub

GitHub fue fundado en 2008 por cuatro desarrolladores que entendieron que Git necesitaba una capa más humana, visual y colaborativa.

👨‍💻

Chris Wanstrath

Cofundador de GitHub y figura clave en el crecimiento inicial de la plataforma.

👨‍💻

Tom Preston-Werner

Cofundador y uno de los impulsores de la idea original de GitHub.

👨‍💻

P. J. Hyett

Cofundador, participó en el desarrollo inicial y expansión de la plataforma.

👨‍💻

Scott Chacon

Cofundador y autor de materiales clave para enseñar Git a desarrolladores.

Evolución

Qué añadió GitHub a Git

Git ya controlaba versiones. GitHub añadió colaboración, comunidad y una interfaz web para hacer el trabajo en equipo mucho más accesible.

📦

Repositorios online

Los proyectos podían guardarse en la nube y compartirse con otros desarrolladores.

🧵

Issues y Pull Requests

Permitieron debatir errores, proponer cambios y revisar código antes de integrarlo.

👥

Comunidad

Los perfiles, estrellas y contribuciones convirtieron GitHub en una red social técnica.

Microsoft

La compra de 2018

En 2018 Microsoft compró GitHub por 7.500 millones de dólares. Fue una operación histórica porque GitHub ya era una infraestructura crítica del software moderno.

La compra marcó un cambio importante: Microsoft, que durante años había tenido una relación complicada con el software libre, pasó a controlar la mayor plataforma de colaboración de código del mundo.

Actualidad

GitHub hoy

GitHub ya no es solo un lugar donde guardar código. Es una plataforma completa para crear, revisar, automatizar, publicar y colaborar.

Herramienta Para qué sirve Idea sencilla
GitHub Repositories Guardar proyectos Git en la nube. El almacén online del proyecto.
GitHub Pages Publicar webs estáticas desde repositorios. El escaparate web del proyecto.
GitHub Actions Automatizar tareas, pruebas y despliegues. El robot que ejecuta procesos.
GitHub Copilot Asistente de IA para ayudar a programar. El copiloto que sugiere código.

Preguntas para clase

Preguntas clave

Úsalas para cerrar la presentación o para responder si el profesor decide poner a prueba el ecosistema neuronal de la sala.

¿GitHub y Git son lo mismo?

No. Git es el sistema de control de versiones. GitHub es una plataforma online para alojar repositorios Git y colaborar.

¿Qué significa Hub?

Hub significa centro o punto de conexión. GitHub es el centro donde se reúnen proyectos Git.

¿Quién creó Git?

Git fue creado en 2005 por Linus Torvalds para gestionar el desarrollo del kernel de Linux.

¿Quién fundó GitHub?

GitHub fue fundado en 2008 por Chris Wanstrath, Tom Preston-Werner, P. J. Hyett y Scott Chacon.

¿Para qué sirve GitHub Pages?

Sirve para publicar webs estáticas directamente desde un repositorio de GitHub.
